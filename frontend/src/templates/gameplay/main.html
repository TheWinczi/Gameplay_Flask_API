Hello!

My name is Maciej Winczewski and I have created this application for playing games with friends in easier way.

Application I have created allows You to create games, adding players to them and controlling their score on any game You created.

Thanks to created API and web page it is really easy to manage whole application objects.

If you want to create new game just go to Games page (available on top of the page) and click on Create New Game.

If You want to create new player just go to Players page (available on top of the page) and click on Create New Player.

If You want to add new player to the game just go to the created game details page and click on Add New Player.
